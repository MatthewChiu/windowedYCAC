# Windowed YCAC

This version of the Yale Classical Archives Corpus is comprised of overlapping windows each separated by one quarter note. Windows contain pitch and pitch-class vectors (pc0, pc1, ..., pc11), where each value is the frequency of the pitch/pitch-class within the window.
- Folders are sorted by window length (each composer is represented within each folder).
- Each pitch-class window is a row in a csv file.
- Rows also contain key-finding information (Krumhansl-Kessler and IQ algorithms).
